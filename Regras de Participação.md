# Regras de Participação – Hackathon

## 1. Participação
- **Aberto para todos:** individualmente ou em equipes de até 4 pessoas.
- **Inscrição obrigatória:** É necessário se inscrever no formulário oficial e participar do palco de abertura.

## 2. Entrega dos Projetos
- **Repositório:** Os projetos devem ser enviados em branch própria no GitHub do hackathon.
- **Arquivo conceito.md:** Cada branch deve conter um arquivo `conceito.md` com as seguintes informações:
  - **Título**
  - **Descrição**
  - **Tecnologias usadas**
  - **Instruções para teste ou cenários de validação:**
    - Passo a passo de uso
    - Cenários
    - Metodologia
    - Resultados esperados
    - Obstáculos identificados
- Caso submeta o projeto através do formulário, lembre-se de deixar o link para o projeto disponível e aberto.

Essas instruções devem permitir que a equipe de avaliação consiga validar a proposta — seja ela com código ou não.

## 3. Código de Conduta
- **Respeito à comunidade:** Obrigatório.
- **Proibido:**
  - Discriminação
  - Assédio
  - Comportamento agressivo (resulta em desclassificação)
- **Discussões políticas:** Devem se manter alinhadas ao tema do evento.
- **Regras do servidor:** Respeitar todas as regras do servidor da Soberana.

## 4. Prazos
- **Inscrições:** até 28/11/2025 às 22:00.
- **Submissão de projetos:** até 30/11/2025 às 22:00.
- **Resultados:** 14/12/2025, com apresentação dos destaques.

## 5. Critérios de Validação
- **Tema:** O projeto precisa atender ao tema: Soberania Digital e Segurança de Dados.
- **Entregas possíveis:**
  - Provas de conceito
  - Estudos
  - Protótipos funcionais

- **Formulário de Inscrição:** [Inscreva-se](https://cpdd.com.br/hackathon/)
- **Exemplos de projetos:** [Acesse](https://github.com/Coletivo-Popular-Design-Desenvolvimento/hackathon-cpdd-soberana/blob/main/Exemplos%20de%20projetos.md)
- **Critérios de revisão:** [Acesse](https://github.com/Coletivo-Popular-Design-Desenvolvimento/hackathon-cpdd-soberana/blob/main/Crit%C3%A9rios%20de%20Revis%C3%A3o.md)

Apenas projetos entregues no prazo recebem certificado de participação.
